# Dog Poop – HTML5 Jump'n'Run Spiel

**Dog Poop** ist ein kleines, spaßiges Jump'n'Run-Spiel, erstellt mit **microStudio**. Das Spiel läuft komplett im Browser und benötigt keine Installation.  

**Dog Poop** is a small, fun jump'n'run game created with **microStudio**. It runs entirely in the browser and requires no installation.

---

## Spielziel / Goal
Spring über die Hindernisse (Haufen) und sammle Punkte, indem du so lange wie möglich überlebst. Mit jedem 10. Punkt wird das Spiel schneller!  
Jump over obstacles (poops) and collect points by surviving as long as possible. Every 10 points, the game speed increases!

---

## Steuerung / Controls
- **SPACE** – Springen / Spiel starten / Jump / Start game  
- **ENTER** – Neustart nach Game Over / Restart after Game Over  

---

## Spiel starten / How to Play
1. Repository herunterladen oder klonen / Clone the repository:
   ```bash
   git clone https://github.com/dein-username/dog-poop.git
2. index.html im Browser öffnen (Chrome, Firefox oder Edge empfohlen) / Open index.html in a browser (Chrome, Firefox, or Edge recommended).

3. Fertig – das Spiel läuft direkt im Browserfenster / Done – the game runs directly in your browser.

Hinweis / Note: Das Spiel läuft offline im Browser. Keine Installation erforderlich.
Note: The game runs offline in the browser. No installation required.
